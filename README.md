Maintenance Cost Sensitivity Analysis

Overview

This project demonstrates how simple linear regression (SLR) can be used to analyse cost behaviour.

The focus is on comparing:

- January (lower variable cost)
- October (higher variable cost)

Key Concept

Regression model:
y = a + bx

- a = fixed cost
- b = variable cost per machine hour

Insight

Although both months show strong linear relationships, their slopes differ:

- January: lower sensitivity to activity
- October: higher sensitivity to activity

This means costs increase more rapidly in October as production rises.

Features

- Interactive Plotly chart
- Machine hours slider
- Real-time cost comparison
- Visual demonstration of slope differences

Conclusion

Segmented regression provides deeper insight than aggregated models, revealing differences in cost structure and operational efficiency.
